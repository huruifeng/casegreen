About

1. Introduction

CaseLand is a non-profit project providing reference statistics of US visa processing for TOTALLY FREE.
CaseLand makes data-driven analysis to result in a reliable prediction of your visa application journey.
We dedicated to track and predict case status as good as possible, based on case processing data from USCIS and user inputs, but the results are not guaranteed

We would like to build an information sharing community for everyone, especially for all the visa applicants. Everyone can access the website, query data, and share information here. This will be a FREE land for all the users.

Meanwhile, all the supports for running this project will be coming from you, If you like, you may contribute your generosity to make this project better and better!
